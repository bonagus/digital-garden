## Masalah

Saya sering beralih konteks antara terminal dan aplikasi task manager (Notion, Todoist). Butuh tool sederhana yang bisa dijalankan langsung dari terminal tanpa meninggalkan workflow coding.

## Solusi

Membuat CLI task manager minimalis yang menyimpan data di file JSON lokal. Operasi CRUD sederhana: `task add`, `task done`, `task list`, `task rm`.

## Teknologi

- **Node.js** — runtime
- **TypeScript** — type safety
- **Commander.js** — CLI framework untuk parsing arguments
- **Chalk** — terminal colors untuk output yang readable

## Fitur

- ✅ Add, complete, delete tasks dari terminal
- ✅ Priority levels (low, medium, high) dengan color coding
- ✅ Filter tasks by status dan priority
- ✅ Data persists di `~/.tasks.json`
- ✅ Tab completion untuk shell (bash/zsh)

## Tantangan

1. **Cross-platform paths** — `~/.tasks.json` harus resolve ke home directory yang benar di Windows, macOS, dan Linux.
2. **Terminal colors** — beberapa terminal emulator tidak support semua ANSI codes, jadi perlu fallback graceful.

## Pelajaran

1. **Commander.js** sangat mudah dipakai — dari nol ke working CLI dalam hitungan jam.
2. **JSON sebagai database** cukup untuk personal tool — tidak perlu SQLite untuk data sederhana.
3. **Menulis tool sendiri** mengajarkan lebih banyak daripada membaca dokumentasi tool orang lain.
